# 지뢰찾기 온라인
## :video_game: 주요 기능
* 웹에서 지뢰찾기를 즐기고 다른 사람과 기록을 경쟁할 수 있습니다
* 게임 기록을 저장하고 리플레이를 볼 수 있습니다
* 다른 사람들과 실시간으로 소통할 수 있습니다
